# MemMem

Run instructions:

Go to: PATH/mem-finalmem/src
Run: python3 main.py "path to input folder"

Ouput txt files will be produced in the input folder.
Segmented lines with bounding boxes and labels generated by the RCNN can be found in PATH/mem-finalmem/src/network/Data/RBA/Outputs


Dependency list:
* Python3
* python-opencv
* numpy
* scipy
* tqdm
* tensorflow
* cython
* pillow
* easydict
* matplotlib
* pyyaml
